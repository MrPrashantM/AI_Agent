# AI Voice Agent

OpenAI GPT + Deepgram se bana **Real-time Voice AI Agent**.

Users apne khud ke API keys daal kar use kar sakte hain.

## ✨ Features
- Real-time Speech to Text (Deepgram)
- AI Response (OpenAI GPT)
- Text to Speech
- Clean Web Interface

## 🛠 Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **AI**: OpenAI GPT
- **Voice**: Deepgram API

## 🚀 How to Run

1. Repository clone karo
2. `voice-ai-agent.html` file ko browser mein kholo
3. Apna **OpenAI API Key** aur **Deepgram API Key** daalo
4. "Start Listening" button dabao

## ⚠️ Note
- API keys client-side use ho rahi hain (sirf testing ke liye)
- Production mein backend use karna recommended hai
